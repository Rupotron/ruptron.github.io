Introduction

Python is a versatile and easy-to-learn programming language that has gained immense popularity in recent years. Its clean syntax, readability, and extensive libraries make it suitable for a wide range of applications, from web development to data science and machine learning.

What you should already know

While not strictly necessary, having a basic understanding of programming concepts can be helpful. However, even if you're completely new to programming, Python's simplicity makes it a great starting point.

Hello world

Let's start with the classic "Hello, world!" program.

print("Hello, world!") The print() function is used to display the message on the console.
Variables

Variables are used to store data. In Python, you can declare variables without specifying their data type.

name="Alice" age = 30 is_student = True
Data Types

In Python, variables are used to store data. Unlike many other languages, Python does not require you to explicitly declare the data type of a variable. The interpreter automatically determines the type based on the assigned value.

Common Data Types
  • Numbers: Integers (e.g., 10), floating-point numbers (e.g., 3.14), and complex numbers.
  • Strings: Integers (e.g., 10), floating-point numbers (e.g., 3.14), and complex numbers.
  • Lists: Ordered collections of elements (e.g., [1, 2, 3]).
  • Tuples: Immutable ordered collections of elements (e.g., (1, 2, 3)).
  • Dictionaries: Unordered collections of key-value pairs (e.g., {"name": "Alice", "age": 30}).
  • Sets: Unordered collections of unique elements (e.g., {1, 2, 3}).
  • Booleans: True or False values.
  • None: A special value representing the absence of a value.
Type Conversion

You can convert between different data types using built-in functions like int(), float(), str(), list(), tuple(), dict(), and set().

x = 10 y = 3.14 z = "Hello" # Type conversion x_str = str(x) # Convert integer to string y_int = int(y) # Convert float to integer z_list = list(z) # Convert string to list
Operators

Python supports various operators for arithmetic, comparison, logical operations, and more.

Arithmatic Operators:
  • +: Addition.
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder)
  • //: Floor division (integer division)
  • **: Exponentiation
Comparison Operators:
  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to
Logical Operators:
  • and: Logical AND
  • or: Logical OR
  • not: Logical NOT
Control Flow

Python provides control flow statements to control the execution of your code.

if...else Statements
if condition: # Code to execute if condition is true else: # Code to execute if condition is false
for Loops:
for item in iterable: # Code to execute for each item
while Loops:
while condition: # Code to execute as long as condition is true
break and continue Statements:
  • break: Terminates the loop early.
  • continue: Skips the current iteration and proceeds
Functions

Modules
Python's modular architecture allows you to organize your code into modules. You can import modules using the import statement. import math result = math.sqrt(25) print(result)
Object-Oriented Programming (OOP)

Python is an object-oriented programming language, which means you can create objects that encapsulate data and behavior.

class MyClass: def __init__(self, name): self.name = name def greet(self): print("Hello, " + self.name + "!") obj = MyClass("Alice") obj.greet()
